## ✨ Features

### 🎨 Design & UI
- **Windows 11 Fluent Design** - Modern, clean interface with smooth animations
- **Mica Effect** - Translucent backgrounds with backdrop blur
- **Dark Mode** - Built-in dark mode with system preference detection
- **Light Mode** - Clean light theme for daytime reading
- **Sepia Mode** - Warm, eye-friendly reading experience
- **Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Sticky Header & Navigation** - Always accessible controls

### 📖 Manga Management
- **Custom Post Types** - Dedicated post types for Manga and Chapters
- **Genre Taxonomy** - Organize manga by genres
- **Status Taxonomy** - Track manga status (Ongoing, Completed, Hiatus, Cancelled)
- **Manga Details** - Author, Artist, Year information
- **Featured Image Support** - Manga covers with custom image sizes
- **Chapter Connection** - Link chapters to their parent manga

### 📱 Full-Screen Manga Reader
- **Dual Reading Modes**
  - *Paged Mode* - Traditional page-by-page reading
  - *Long Strip Mode* - Continuous vertical scrolling
- **Image Settings**
  - Image Fit: Contain / Cover / Original
  - Scale: 100% / Auto / Fit to Width
  - Zoom controls (50% - 250%)
- **Background Modes**
  - Dark Mode (default)
  - Light Mode
  - Sepia Mode
- **Navigation Options**
  - Click to advance
  - Buttons only
  - Keyboard controls (← → arrows, F for fullscreen, S for settings)
- **Chapter Selector** - Jump between chapters easily
- **Progress Bar** - Track reading progress
- **Page Indicator** - See current position
- **Settings Persistence** - All preferences saved to localStorage
- **Fullscreen Support** - Native browser fullscreen API

### 📁 Folder Import System
- **Local Folder Import** - Import chapters directly from `/manga/` directory
- **Bulk Import** - Import multiple chapters at once
- **Auto Manga Creation** - Automatically creates manga entries from folder names
- **Duplicate Prevention** - Tracks imported chapters to prevent duplicates
- **Image Upload** - Supports JPG, JPEG, PNG, GIF, WebP formats
- **Cache Management** - Clear import cache when needed
- **Note** - If you are using a free shared hosting this function may use a lot of CPU and slow down your site/rate limit

## 📋 System Requirements

- WordPress 5.0 or higher
- PHP 7.4 or higher
- MySQL 5.6 or higher
- Modern web browser with JavaScript enabled

## 🚀 Installation

### Method 1: Upload via WordPress Admin
1. Download the theme zip file
2. Go to **Appearance → Themes → Add New → Upload Theme**
3. Choose the zip file and click **Install Now**
4. Click **Activate**

### Method 2: FTP Installation
1. Extract the theme folder
2. Upload to `/wp-content/themes/` directory
3. Go to **Appearance → Themes** and activate
